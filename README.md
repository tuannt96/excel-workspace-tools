# Excel Workspace Tools

A small Excel task pane add-in project built with Office Add-ins and Office.js.

## Purpose

This project explores how to build lightweight tools inside Excel using a task pane interface. The add-in starts with a simple workbook interaction and can later be expanded into utilities for formatting, checking, organizing, or summarizing worksheet data.

## Features

* Excel task pane interface
* Office.js workbook interaction
* Local development workflow
* Simple command button for testing Excel actions
* Structure ready for future worksheet utilities

## Tech stack

* Office Add-ins
* Office.js
* Excel
* Node.js
* TypeScript
